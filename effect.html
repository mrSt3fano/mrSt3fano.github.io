UseEffect

El Hook useEffect es un Hook de efecto que nos permite ejecutar tareas secundaria
dentro de nuestros componentes funcionales, por ejemplo, podemos emular los métodos
de ciclo de vida, sin la necesidad de utilizar componentes de clase.

                
UseEffect con parámetro vacío

                
                    
                    const [data, setData] = useState(null);
        
                    useEffect(() => {
                        fetch('https://api.example.com/data')
                          .then(response => response.json())
                          .then(data => setData(data));
                      }, []);
                  
                  
UseEffect con parámetro: 

                  
                    useEffect(() => {
                            // efecto
                            }, [data]);
                      
                      
En este caso, cuando el valor de data cambia, el efecto se vuelve a ejecutar
