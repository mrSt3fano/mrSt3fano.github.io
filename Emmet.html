Sobre Emmet

Emmet es un plugin que existe para la mayoría de los editores de texto comunes, pensado para mejorar el flujo de trabajo, la principal funcionalidad de Emmet es que no necesita instalación, este plugin está disponible de forma predeterminada para VS Code y facilita la generación de código HMTL y la apertura de etiquetas.

Emmet es una potente herramienta de expansión de código que facilita la escritura de código HTML y CSS de manera más rápida y eficiente. Fue creado por Sergey Chikuyonok y lanzado en 2008 bajo el nombre original "Zen Coding". Posteriormente, en 2012, cambió su nombre a Emmet.
Con Emmet, se puede escribir código más rápidamente utilizando abreviaturas y combinaciones de teclas, y Emmet se expandirá automáticamente en código HTML o CSS completo.
