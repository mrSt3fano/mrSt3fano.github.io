            const [state, dispatch] = useReducer(reducer, { age: 42 });

Dispatch

          
La función dispatch devuelta por useReducer te permite actualizar el estado 
 a un valor diferente y activar una nueva renderización.

          const [state, dispatch] = useReducer(reducer, { age: 42 });
          
            function handleClick() {
            dispatch({ type: 'incremented_age' });
            // ...
            }
          
            

            
    Parámetro
                
  1. action: La acción realizada por el usuario.
