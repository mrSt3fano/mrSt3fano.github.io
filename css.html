CSS

CSS, Cascading Style Sheets, Hojas de Estilo en Cascada Es un lenguaje utilizado para la presentación (diseño o aspecto visual) del contenido de una página web. Utilizando CSS daremos estilo a cualquier tag de HTML. Se trata de una tecnología utilizada para dotar de cualidades visuales y estéticas a una página web.

    Evolución Histórica

Cascading Style Sheets (CSS) ha experimentado una evolución significativa desde su introducción, contribuyendo de manera fundamental al diseño y presentación de páginas web. Aquí hay un breve resumen de la historia de CSS:

  1. Inicios (1996): CSS fue propuesto por Håkon Wium Lie y Bert Bos en 1994.
  2. CSS2 (1998)
  3. Estándares (2000-2007)
  4. CSS3 (2001 en adelante)
  5. Adopción generalizada (2010 en adelante)
  6. Flexbox y Grid (2011-2017)
  7. CSS4 y el futuro

EJE PRINCIPAL

El eje principal está definido por flex-direction, que posee cuatro posibles valores:

  • row
  • row-reverse
  • column
  • column-reverse

EJE CRUZADO

El eje cruzado va perpendicular al eje principal, y por lo tanto si flex-direction (del eje principal) es row o row-reverse el eje cruzado irá por las columnas.

Si el eje principal es column o column-reverse entonces el eje cruzado corre a lo largo de las filas.
