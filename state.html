React - UseState

UseState: Hook de React para agregar una variable de estado al componente.

Ejemplo 1:

        
            const [age, setAge] = useState(0);
            setAge[age+1];

age: Variable de estado que mantiene los datos entre renderizaciones.
setAge: Función que setea el estado y provoca una nueva renderización.
0: Valor inicial de la variable de estado.

Ejemplo 2:

        
            const [form, setForm] = useState({
                firstName: 'Stefano',
                lastName: 'Soto',
                email: 'stefano13soto@gmail.com',
              });
            
                        <input
            value={form.firstName}
            onChange={e => {
                setForm({
                ...form,
                firstName: e.target.value
                });
            }}
            />

Usos comunes:

  • Manejo de variables que cambian con el tiempo.
  • Actualizar y renderizar componentes basados en eventos o interacciones del usuario.
  • Controlar el estado de formularios y manejar eventos como clics o envíos.
